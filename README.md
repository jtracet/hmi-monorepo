# HMI Editor — Руководство пользователя

## Интерфейс редактора

### Холст

Центральная рабочая область. Сюда перетаскиваются элементы из левой панели.

- **Перемещение по холсту** — зажмите `ЛКМ` и тяните мышью
- **Переход в режим масштабирования** — двойное нажатие`ЛКМ`
- **Выделение нескольких элементов** — зажмите и потяните мышью по пустой области холста
- **Удалить** — `Delete`

---

### Левая панель

#### Переключатель режимов

В верхней части левой панели находятся две кнопки:

- **Тестирование** — режим тестирования. Элементы реагируют на клики, отображают данные из привязанных переменных.
- **Редактирование** — режим проектирования. Элементы можно перемещать, изменять размер, настраивать свойства. Клики по элементам не вызывают их действий.

#### Библиотека элементов

Ниже переключателя — список доступных элементов, сгруппированных по категориям. Каждую категорию можно раскрыть кликом. Внутри категорий есть подгруппы **Controls** (управляющие элементы) и **Indicators** (индикаторы).

Чтобы добавить элемент на холст — **перетащите его** из панели на нужное место.

#### Кнопки внизу

- **Сохранить (JSON)** — экспортирует текущий экран в файл `.hmi.json`. Файл содержит все элементы, их свойства, привязки к переменным и настройки вида.
- **Загрузить...** — открывает диалог выбора файла `.hmi.json` и загружает сохранённый экран на холст.

---

### Страницы (Pages)

Один файл `.hmi.json` может содержать несколько страниц. Каждая страница — отдельный холст со своим набором элементов.

#### Панель страниц

Расположена внизу редактора. Отображает вкладки всех страниц текущего проекта.

- **Переключение** — кликните по вкладке нужной страницы
- **Добавить страницу** — кнопка `+` справа от вкладок
- **Переименовать** — двойной клик по вкладке, введите новое имя, `Enter` для подтверждения или `Escape` для отмены
- **Удалить страницу** — кнопка `×` на вкладке (доступна если страниц больше одной)

#### Сохранение и загрузка

При нажатии **Сохранить (JSON)** в файл записываются все страницы сразу. При **Загрузить...** восстанавливаются все страницы из файла, активной становится та, которая была активной при сохранении.

#### Просмотр во Viewer

Viewer также поддерживает многостраничные файлы. Панель страниц отображается внизу — переключение между страницами работает так же, как в редакторе.

---

### Правая панель

Появляется при выделении элемента.


#### Свойства

Здесь настраиваются параметры выбранного элемента:

- **Label** — текст подписи под элементом
- **Label font size** — размер шрифта подписи
- **Шрифт** — гарнитура шрифта (применяется и к подписи, и к числу внутри элемента)
- **Насыщенность шрифта** — Normal, Bold, 900

Остальные поля зависят от типа элемента и описаны в разделе «Элементы» ниже.

#### Входы / Выходы (привязка к переменным)

Каждый элемент может быть привязан к переменным бэкенда:

- **Входы (read)** — переменная, значение которой элемент читает и отображает
- **Выходы (write)** — переменная, в которую элемент записывает значение при взаимодействии

Выберите нужную переменную из выпадающего списка. После привязки в правом верхнем углу элемента появится **зелёный кружок** — индикатор активной привязки.

## Элементы

### Numeric Input

**Категория:** Numeric -> Controls

Поле для ввода числового значения. Используется для отправки числа в систему.

**Свойства в правой панели:**
- `value` — текущее значение
- `fontSize` — размер шрифта числа внутри поля

**Поведение:**
- В режиме **редактирования** — значение меняется через правую панель
- В режиме **тестирования** — кликните по элементу, поле подсветится синим, введите число с клавиатуры (поддерживаются целые и дробные через `.`), нажмите `Enter` для подтверждения или `Escape` для отмены

> Если после подтверждения значение не отобразилось в системе — кликните по элементу повторно и нажмите `Enter` ещё раз.

---

### Numeric Control

**Категория:** Numeric -> Controls

Числовое поле с кнопками `<-` и `->` для пошагового изменения значения.

**Свойства в правой панели:**
- `value` — текущее значение
- `step` — шаг изменения при нажатии кнопки
- `fontSize` — размер шрифта числа

**Поведение:**
- В режиме **редактирования** — значение и шаг меняются через правую панель
- В режиме **интерактива**:
  - Клик по `<-` — уменьшает значение на шаг
  - Клик по `->` — увеличивает значение на шаг
  - Клик по центральному полю — открывает инлайн-редактор (как в Numeric Input)

> Если после подтверждения значение не отобразилось в системе — кликните по центральному полю повторно и нажмите `Enter` ещё раз.

---

### Numeric Indicator

**Категория:** Numeric -> Indicators

Поле только для отображения числа. Не принимает ввод от пользователя.

**Свойства в правой панели:**
- `fontSize` — размер шрифта
- `precision` — количество знаков после запятой

**Поведение:** отображает значение привязанной переменной. До получения первого значения показывает `--`.

---

### Slide Switch

**Категория:** Boolean -> Controls

Переключатель вкл/выкл в виде ползунка.

**Свойства в правой панели:**
- Раздел **«Управление переключателем»** — показывает текущее состояние и позволяет переключить его вручную через правую боковую панель

**Поведение:**
- В режиме **Тестирования** — клик переключает состояние с анимацией. Синий цвет — включено, серый — выключено.

---

### LED Indicator

**Категория:** Boolean -> Indicators

Круглый световой индикатор.

**Свойства в правой панели:**
- `onColor` — цвет в состоянии «включено»
- `offColor` — цвет в состоянии «выключено»

**Поведение:** меняет цвет в зависимости от значения привязанной переменной. В режиме интерактива также реагирует на клик (переключает состояние вручную).

---

### Tank

**Категория:** Numeric -> Indicators

Визуальный индикатор уровня в виде резервуара с заливкой. Размер фиксированный (80×150 px), изменяется масштабированием на холсте.

**Свойства в правой панели:**
- `minValue` / `maxValue` — диапазон значений
- `fillColor` — цвет заливки
- `emptyColor` — цвет пустой части
- `borderColor` — цвет рамки
- `showValue` — показывать ли числовое значение внутри
- `valueFontSize` — размер шрифта числа

**Поведение:** уровень заливки пропорционален значению переменной в диапазоне min–max.

---

### Time Graph

**Категория:** Graph

График значения переменной во времени. Новые точки появляются справа, старые сдвигаются влево. Шкала времени фиксированная.

**Свойства в правой панели** (раздел «Шкалы графика»):
- **Макс. значение (Y)** — верхняя граница оси значений
- **Шаг по Y** — интервал между отметками на оси Y
- **Шаг по времени (сек)** — как часто добавляется новая точка
- **Точек на шкале времени** — количество точек на оси X (ширина окна наблюдения). Всегда фиксировано — до поступления данных точки равны 0.

**Поведение:** график активен только в режиме **тестирования** и только при наличии привязанной переменной.

---

### Line

**Категория:** Decorations

Декоративная линия. Не привязывается к переменным.

**Свойства в правой панели:**
- `stroke` — цвет линии
- `strokeWidth` — толщина линии

---

### Image

**Категория:** Decorations

Изображение (PNG или JPEG). Не привязывается к переменным.

**Добавление:** перетащите элемент Image на холст — откроется диалог выбора файла. Поддерживаются форматы `.png` и `.jpg`.
